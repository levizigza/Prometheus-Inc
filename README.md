# Prometheus

Website for **Prometheus** — where robotics, architecture, and humanity evolve together in harmony.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Logo

Place your Prometheus logo (the flame-in-block image) at **`assets/logo.png`** so it appears in the header, hero, and footer. If the file is missing, those areas will show a placeholder.

## Structure

- `index.html` — Single-page layout: hero, vision, robotics, solar punk cities, philosophy, contact
- `styles.css` — Solar punk / mythic theme (warm ambers, deep greens, Outfit + Playfair Display)
- `main.js` — Header scroll state and mobile menu toggle

## Contact

Update the contact section in `index.html` with your real email or link to a contact form.
